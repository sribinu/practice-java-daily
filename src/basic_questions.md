## Basic Questions

1. Write a Java program to print “Hello, Java Developer!”.
2. Take user input for name and age, then print a greeting message.
3. Read two numbers and print their sum.
4. Swap two numbers without using a third variable.
5. Calculate simple interest for given P, T, R.
6. Calculate area and circumference of a circle using final double PI.
7. Convert Fahrenheit temperature to Celsius.
8. Print ASCII value of a given character.
9. Take two integers from command-line arguments and print their sum.
10. Multiply two floating-point numbers and display the result.
11. Demonstrate implicit and explicit type casting.
12. Print min and max values of all primitive data types using wrapper constants.
13. Convert an integer to string and string to integer.
14. Take a boolean variable and toggle its value.
15. Perform +, −, \*, /, % operations on two integers.
16. Compare two numbers using relational operators and print results.
17. Demonstrate logical operators (\&\&, ||, !) with clear output.
18. Demonstrate difference between pre-increment and post-increment.
19. Find the largest of three numbers using the ternary operator.
20. Perform bitwise AND, OR, XOR, complement, left shift, and right shift.
21. Demonstrate instanceof with class inheritance.
22. Show how +=, -=, \*=, /= operators work.
23. Check whether a number is even or odd using if statement.
24. Determine if a number is positive, negative, or zero.
25. Find the largest among three numbers using nested if.
26. Check if a character is a vowel or consonant using if-else.
27. Input marks and print grade (A, B, C, etc.) using if-else if.
28. Check if a year is a leap year using conditional statements.
29. Build a simple calculator using switch statement.
30. Display weekday name based on number (1–7) using switch.
31. Calculate sum of first N natural numbers using for loop.
32. Find factorial of a number using for loop.
33. Reverse digits of a number using while loop.
34. Check whether a number is palindrome using while.
35. Count number of digits in a number using loop.
36. Print multiplication table for a given number.
37. Find sum of even and odd numbers separately up to N.
38. Print Fibonacci series up to given terms.
39. Check if a number is prime.
40. Check if a number is Armstrong or not.
41. Check if a number is perfect or not.
42. Check if a number is strong (sum of factorial of digits = number).
43. Find the sum of digits of a given number.
44. Calculate power of a number without using Math.pow().
45. Find sum of series 1² + 2² + 3² + … + N².
46. Check if a number’s square ends with the same number.
47. Check if a number is neon (sum of digits of its square equals the number).
48. Check if a number is spy (sum of digits = product of digits).
49. Check whether a number is xylem or phloem number.
50. Check if a number is special or not.
51. Check if a number is sunny or not.
52. Check if a number is happy or not.
53. Find factorial of a number using for recursion.
54. Print Fibonacci series up to given term using recursion.
